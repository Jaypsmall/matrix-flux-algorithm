# Algoritmo de Flujo Asimétrico y Absorción Geométrica

## Descripción de la Tesis
Este repositorio contiene un ejercicio teórico y abstracto de gestión de flujos de capital aplicado a matrices de tres variables con retorno proporcional (3:1).

## Lógica del Algoritmo
El sistema evalúa el comportamiento de un "muelle elástico" distribuido en tres frentes simultáneos. Al ocurrir un evento, la variable ganadora reduce su exposición al mínimo absoluto, mientras que las variables rezagadas duplican su volumen geométricamente para forzar una compensación matemática en eventos futuros.

## Análisis de Riesgo
El script actúa como un comando de laboratorio para estudiar la resistencia del capital frente a la varianza y determinar el umbral exacto donde la progresión geométrica supera la liquidez del operador o del sistema emisor.
